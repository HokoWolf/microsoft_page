Основні дати в історії компанії

1975

Друзі по навчанню Біл Гейтс та Пол Ален заснували компанію Microsoft. Першим їх продуктом був інтерпритатор мови Basic.

1980

Microsoft підписала договір з IBM на розробку оперційної системи MS-DOS

1983

Реліз першої версії MS Word.

1985

Реліз першої Excel. Версія була призначена тільки для комп'ютерів Macintosh.

1995

Реліз Windows 95 з оновленим інтерфейсом на кнопкою "Пуск".

1995

Біл Гейтс став найбагатшою людиною у світі та зберігав цей статус до 2017 рок.
